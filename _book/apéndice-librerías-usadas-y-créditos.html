RECUPERACIÓN, EXTRACCIÓN Y CLASIFICACIÓN DE INFORMACIÓN DE SABER UCV

Apéndice: Librerías Usadas y Créditos

  1. Las librerías de R que usa la aplicación web son:

    1. shinydashboard: sirve para gener un formato de dashboard en la aplicación (Chang y Borges Ribeiro 2021).

    2. shinydashboardPlus: asiste a la librería “shinydashboard” en configuraciones y funcionalidades adicionales (Granjon 2021).

    3. shinycssloaders: muestra un loader cuando la aplicación de está cargando (Sali y Attali 2020).

    4. pool: gestor de conexión mediante un pool a la base de datos (Cheng, Borges, y Wickham 2023).

    5. RPostgres: carga el protocolo de conexión a postgreSQL (Wickham, Ooms, y Müller 2023).

    6. DBI: permite la definición en R de querys en lenguaje SQL (, Wickham, y Müller 2022).

    7. dplyr: herramienta que facilita la manipulación de las dataframes que sirven de insumo para representar los resultados (Wickham et al. 2023).

    8. tidyr: manipulación de la dataframe que contiene los resultados (Wickham, Vaughan, y Girlich 2023).

    9. dbplyr: bajo la sintaxis de “dplyr” permite estructurar los querys (Wickham, Girlich, y Ruiz 2023).

    10. DT: wrapper de la librería “DataTables” de Javascript que permite generar tablas interactivas en html que muestran los resultados de los querys (Xie, Cheng, y Tan 2023).

    11. udpipe: paquete que permite manejar textos bajo los principios del framework “Universal Dependencies” y en la aplicación se usa para generar la estructura de datos que representa la coocurrencias (Wijffels 2023).

    12. ggraph: permite realizar la visualización estática mediante grafos de las coocurrencias (Pedersen 2022).

    13. visNetwork: permite realizar la visualización dinámica mediante grafos de las coocurrencias. Es un wrapper de la libraría Vis.Js.

    14. apexcharter: generar gráficos interactivos y es un wrapper de la librería “ApexCharts.Js” (Perrier y Meyer 2023).

    15. scales: se usa para reescalar los arcos que unen los nodos en las coocurrencias (Wickham y Seidel 2022).

    16. fontawesome: usado para mostrar íconos en los selectores y ventanas (Iannone 2023).

    17. stringr: manipulación y manejo de textos (Wickham 2022b).

    18. shinyBS: permite añadir los tooltips (Bailey 2022).

    19. shinyalert: crea un popup al ingresar a la aplicación.

    20. shinyWidgets: herramienta para manejar interactividades (Perrier, Meyer, y Granjon 2023).

    21. config: configurar variables globales (Allaire 2020).

  2. Librerías usadas por el Contenedor “R Servicios”:

    1. httr2: ejecutar consultas HTTp y procesar las respuestas (Wickham 2023).

    2. pool: gestor de conexión mediante un pool a la base de datos (Cheng, Borges, y Wickham 2023).

    3. RPostgres: carga el protocolo de conexión a postgreSQL (Wickham, Ooms, y Müller 2023).

    4. DBI: permite la definición en R de querys en lenguaje SQL (, Wickham, y Müller 2022).

    5. odbc: soporte para conexiones a manejadores de base de datos (Hester, Wickham, y Gjoneski 2023).

    6. dplyr: herramienta que facilita la manipulación de las dataframes que sirven de insumo para representar los resultados (Wickham et al. 2023).

    7. tidyr: manipulación de la dataframe que contiene los resultados (Wickham, Vaughan, y Girlich 2023).

    8. dbplyr: bajo la sintaxis de “dplyr” permite estucturar los querys (Wickham, Girlich, y Ruiz 2023).

    9. stringr: manipulación y manejo de textos (Wickham 2022b).

    10. rvest: manipulación y extracción de datos de páginas web (Wickham 2022a).

    11. rlang: caja de herramientas para trabajar con estructuras de R base (Henry y Wickham 2023).

    12. purrr: aplicar programación funcional (Wickham y Henry 2023).

    13. readtext: importar archivos en formato “word” o “pdf” (Benoit y Obeng 2023) .

    14. text.alignment: aplicar el algoritmo “Smith-Waterman” (Wijffels 2022).

    15. spacyr: wrapper para usar el framework Spacy (Benoit y Matsuo 2020).

    16. config: configurar variables globales (Allaire 2020).

    17. tm: ejecutar procesos de minería de texto (Feinerer y Hornik 2023).

    18. quanteda: análisis cuantitativo de textos (Benoit et al. 2018).

  3. Para realizar los diagramas y esquemas representados en esta investigación se uso el software “draw.io” («Security-First Diagramming for Teams» 2023).

  4. Este documento, tanto en su versión PDF como en HTML, se generó con la librería bookdown (Xie 2023).
